# Installation
### Download and change folder
```
cd ~
git clone https://github.com/ve3eab/dotfiles

OR

git clone git@github.com:ve3eab/dotfiles.git

cd dotfiles
```
### Run install
```
./install.sh
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/dotfiles
```

### Run uninstaller
```
./uninstall.sh
```
